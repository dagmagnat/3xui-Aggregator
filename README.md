🚀 3xui-Aggregator

Панель для объединения подписок и управления клиентами сразу на нескольких 3x-ui узлах.

Позволяет создать клиента один раз и автоматически развернуть его на всех выбранных серверах.

⚙️ Возможности
🌐 Управление несколькими 3x-ui узлами
👤 Создание клиента сразу на нескольких серверах
🔁 Автоматическая синхронизация конфигов
📡 Поддержка VLESS + Reality
🔗 Одна общая подписка для клиента
📋 Кнопка копирования подписки
🏷 Красивые названия нод (с флагами стран)
🧠 Автоматическая генерация ссылок
🧾 Настраиваемое название подписки
🔍 Проверка доступности узлов
🗑 Удаление клиента со всех узлов
🧠 Как это работает
Ты добавляешь узлы (сервера с 3x-ui)
Создаёшь клиента
Агрегатор:
создаёт клиента на каждом узле
собирает все конфиги
объединяет их в одну подписку

📦 В итоге клиент получает одну ссылку, внутри которой все серверы.

⚡ Быстрая установка
bash <(curl -fsSL https://raw.githubusercontent.com/dagmagnat/3xui-Aggregator/main/install.sh)
🛠 Что делает установщик
устанавливает Docker
клонирует проект в /opt/3xui-aggregator
создаёт .env
запускает контейнеры
при выборе домена:
автоматически поднимает HTTPS через Caddy
🌍 Режимы установки
1. По IP (простой)
http://IP:3000

✔ быстро
❌ нет HTTPS
❌ могут быть проблемы с копированием подписок

2. По домену (рекомендуется) ✅
https://your-domain.com

✔ HTTPS
✔ корректная работа клиентов
✔ корректное копирование ссылок
✔ красиво и удобно

⚠️ ВАЖНО

👉 Рекомендуется использовать ДОМЕН, а не IP

Почему:

некоторые клиенты (особенно Happ, Clash и т.д.) могут:
не копировать подписку
не обновлять конфиги
браузеры могут блокировать функции (clipboard API)
HTTPS работает корректно только с доменом
⚠️ Ограничения SSL (важно)

Let's Encrypt имеет лимиты:

❗ максимум 5 сертификатов на домен за 7 дней

Если часто переустанавливаешь:

👉 используй новый поддомен:

agg.domain.com
vpn.domain.com
panel.domain.com
⚠️ Конфликт с 3x-ui

Если ставишь на один сервер:

Что	Порт
3x-ui	80 / 443
Aggregator (Caddy)	80 / 443

👉 Решения:

использовать другой сервер (лучше)
или:
агрегатор через IP:3000
3x-ui через домен
