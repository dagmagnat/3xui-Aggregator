# 🚀 3xui-Aggregator

Панель для объединения подписок и управления клиентами сразу на нескольких 3x-ui узлах.

Позволяет создать клиента один раз и автоматически развернуть его на всех выбранных серверах.

---

## ⚙️ Возможности

- 🌐 Управление несколькими 3x-ui узлами  
- 👤 Создание клиента сразу на нескольких серверах  
- 🔁 Автоматическая синхронизация конфигов  
- 📡 Поддержка VLESS + Reality  
- 🔗 Одна общая подписка для клиента  
- 📋 Кнопка копирования подписки  
- 🏷 Красивые названия нод (с флагами стран)  
- 🧠 Автоматическая генерация ссылок  
- 🧾 Настраиваемое название подписки  
- 🔍 Проверка доступности узлов  
- 🗑 Удаление клиента со всех узлов  

---

## 🧠 Как это работает

1. Ты добавляешь узлы (сервера с 3x-ui)  
2. Создаёшь клиента  

Агрегатор:

- создаёт клиента на каждом узле  
- собирает все конфиги  
- объединяет их в одну подписку  

📦 В итоге клиент получает одну ссылку, внутри которой все серверы.
